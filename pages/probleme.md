# Probleme & Lösungen

1. **FiveM crasht beim Laden**:  
    - Folgende Fehler werden hier behoben:
        - `DirectX encountered an unrecoverable error`
        - `FiveM crashes due to NVIDIA GPU drivers.`
        - `FiveM crashes due to AMD GPU drivers.`

    - Mögliche Lösungen:
        - Installiere die neueste Version von DirectX:
          [DirectX End-User Runtime Web Installer](https://www.microsoft.com/de-de/download/details.aspx?id=35)
        - Aktualisiere deine GPU-Treiber:
            - [NVIDIA Treiber Download](https://www.nvidia.com/de-de/drivers/)
            - [AMD Treiber Download](https://www.amd.com/de/support/download/drivers.html)
            - [Intel Treiber Download](https://www.intel.de/content/www/de/de/support/detect.html)

2. **FiveM has encountered an error**:
    - Mögliche Lösungen:
        - Öffne den Rockstar Games Launcher
        - Installiere alle verfügbaren GTA V (legacy) Updates.
        - Lösche den `cache`:
            - Drücke die Win Taste und die R Taste gleichzeitig und gib ein:
            `%localappdata%\fivem\fivem.app\data`
            - Lösche folgende Ordner:
                - `cache`
                - `server-cache`
                - `server-cache-priv`

3. **Das FiveM UI laggt (Smartphone usw)**:
    - Mögliche Lösungen:
        - Aktiviere die `Fix UI Lag` Option in den FiveM Einstellungen. [Tutorial](https://cdn.discordapp.com/attachments/1292840976364339273/1322768340766429194/uJKCcmO.png?ex=687db404&is=687c6284&hm=f4bc412be21c0a202540a7f91861f5ddb3e124db456c8c390b68eca910bd063c&)
